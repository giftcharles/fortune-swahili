üì¶ Quick Installation

1Add GPG Key

Import the repository signing key to verify package authenticity:

curl -fsSL https://giftcharles.github.io/fortune-swahili/public.key.asc | sudo gpg --dearmor -o /usr/share/keyrings/fortune-swahili.gpg

2Add Repository

Add the Fortune Swahili repository to your sources:

echo "deb [signed-by=/usr/share/keyrings/fortune-swahili.gpg] https://giftcharles.github.io/fortune-swahili stable main" | sudo tee /etc/apt/sources.list.d/fortune-swahili.list

3Install Package

Update your package list and install:

sudo apt update sudo apt install fortune-swahili

4Enjoy!

Get your daily dose of Swahili wisdom:

fortune-swahili

Pro tip: Add to your shell profile for daily wisdom when you open a terminal!
