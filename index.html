Fortune Swahili

A Debian package delivering authentic Swahili proverbs to your terminal

Fortune Swahili is a command-line utility that provides access to a curated collection of 5,698 traditional Swahili proverbs. This package integrates seamlessly with your system, offering instant access to centuries of cultural wisdom and linguistic heritage.

üì¶ Quick Installation

1Add GPG Key

Import the repository signing key to verify package authenticity:

curl -fsSL https://giftcharles.github.io/fortune-swahili/public.key.asc | sudo gpg --dearmor -o /usr/share/keyrings/fortune-swahili.gpg

2Add Repository

Add the Fortune Swahili repository to your sources:

echo "deb [signed-by=/usr/share/keyrings/fortune-swahili.gpg] https://giftcharles.github.io/fortune-swahili stable main" | sudo tee /etc/apt/sources.list.d/fortune-swahili.list

3Install Package

Update your package list and install:

sudo apt update sudo apt install fortune-swahili

4Run the Command

Execute the command to display a random Swahili proverb:

fortune-swahili

Optionally, add this command to your shell configuration file (.bashrc or .zshrc) for automatic execution on terminal startup.
